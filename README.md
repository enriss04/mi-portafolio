# Portafolio - Enrique Sánchez Santiago

Proyecto construido con **Next.js 15+** y **TypeScript**, optimizado para despliegue en contenedores mediante el modo **Standalone**.

## 🛠️ Requisitos
- **Docker** y **Docker Compose**
- **Make** (opcional, para usar los comandos simplificados)

## 🚀 Despliegue en Producción

Para poner en marcha el portafolio en tu servidor, simplemente ejecuta el comando de despliegue automatizado:

```bash
make deploy
